## 🚀 Как внести вклад

### 1. Форк репозитория
1. Форкните проект
2. Создайте свою feature-ветку
   ```bash
   git checkout -b feature/amazing-feature
   ```

### 2. Commits
- Используйте понятные commit-сообщения
- Следуйте формату: `тип(область): краткое описание`
  - Типы: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`

### 3. Pull Request
1. Откройте Pull Request
2. Опишите изменения
3. Убедитесь, что прошли все тесты

## 🛠 Стандарты кода
- Используйте TypeScript для фронтенда
- Соблюдайте PEP 8 для Python
- Пишите понятный и чистый код
- Добавляйте комментарии при необходимости
