# 🤝 Contributing to Bonus APP

## 🌈 Как внести свой вклад

Мы рады вашему участию в развитии Bonus APP! Этот документ содержит руководство по внесению вклада в проект.

## 📋 Процесс контрибуции

1. 🍴 Форкните репозиторий
2. 🌿 Создайте новую ветку (`git checkout -b feature/amazing-feature`)
3. 💻 Внесите изменения
4. ✅ Убедитесь, что код прошел все тесты
5. 📝 Сделайте коммит (`git commit -m 'Add some amazing feature'`)
6. 🚀 Запушьте в свой форк (`git push origin feature/amazing-feature`)
7. 🔀 Создайте Pull Request

## 🧪 Тестирование

- Frontend: `npm run test`
- Backend: `pytest`

## 📝 Правила написания кода

### Frontend (React Native)
- Используйте TypeScript
- Следуйте стилю ESLint
- Пишите юнит-тесты для новой функциональности

### Backend (FastAPI)
- Используйте аннотации типов
- Следуйте PEP 8
- Покрывайте код тестами

## 🐛 Сообщения об ошибках

Используйте шаблон:
- Краткое описание
- Шаги воспроизведения
- Ожидаемое поведение
- Фактическое поведение
- Скриншоты (если применимо)

## 📜 Лицензия

Участвуя в проекте, вы соглашаетесь с тем, что ваш код будет распространяться под MIT License.

🌟 Спасибо за ваш вклад!
